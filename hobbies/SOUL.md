## Conventions

- All DIY work uses **metric units** (mm, kg).
- All electrical work uses **12V DC** LED systems with screw-terminal drivers (Pro Plus). Never use proprietary snap-in connectors.
- Wall cabinets: **18mm plywood** with face frames, back-fixing strips (2 points per unit), internal wire routing.
- **Suppliers**: Banggood/Enjoywood (primary tools), Screwfix (primary fixings/materials), Amazon UK (secondary), Solmer.co.uk (LED components). Include exact product name, price (GBP), and URL in recommendations.
- **PARA structure**: active projects in ~/para/1_projects/, each with README, notes, assets. Reference the project management agent at ~/para/ for current status.
- Distinguish between consumer-grade and professional-grade tool tiers.
